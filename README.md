# Mass Effect 5e GM Screen (LaTeX)

A Mass Effect 5e GM screen project built in LaTeX. Feel free to fork and add your own elements or improve the mess that is this code. If you only need the screen grab main.pdf.

## Structure

- `main.tex` - main document
- `preamble.tex` - packages, colors, reusable commands
- `content/` - panel content files
- `assets/` - panel backgrounds and visual assets
